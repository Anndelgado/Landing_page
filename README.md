# Fusion Gourmet – Landing Page

## Project Description

Fusion Gourmet is a modern and elegant landing page designed for a gourmet restaurant.
The website show you featured chefs, and event schedules, with the main goal of users to reserve their spot.

## Technologies Used

* HTML
* CSS
* JavaScript (Only for a basic interactivity, when is in a mobile)

## Project Structure

/project-folder
│
├── gastro.html        # Main HTML structure
├── style.css          # Styles and layout
├── script.js          # Interactive features (menu)
├── /images            # Images and media (gallery, logo, video)
└── README.md          # Project documentation

## Features

* Responsive design (mobile & desktop)
* Navigation menu with a icon
* Event schedule displayed in a table
* About Us section 
* Image and video gallery
* Reservation form for booking experiences
* Styled footer with contact and social links

## Responsive Design

The layout adapts to different screen sizes using CSS media queries:

* Mobile devices (≤768px)
* Desktop screens (>1024px)


##  How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `gastro.html` in your browser

## Purpose

This project was created as part of a frontend development assignment focused on HTML5 and CSS3.
It demonstrates the use of semantic structure, responsive design, and modern UI/UX practices.
